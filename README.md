# Text Summarizer using NLP

This project is a simple Text Summarization application built using Python and NLP techniques. The model generates short summaries from long text or dialogues. It also includes a basic web interface for user interaction.

## Features

- Automatic text summarization
- NLP-based preprocessing
- Simple and clean UI
- Uses trained summarization model
- Supports long text input

## Technologies Used

- Python
- TensorFlow / Transformers
- Flask
- HTML
- NLP

## Project Files

- `app.py` → Fast api application
- `index.html` → Frontend UI
- `text_summarizer.ipynb` → Model training and testing notebook
- `samsum-train.csv` → Training dataset
- `samsum-test.csv` → Testing dataset
- `samsum-validation.csv` → Validation dataset
